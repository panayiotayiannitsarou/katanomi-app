# 📚 Εργαλείο Κατανομής Μαθητών

Αυτή είναι μια εφαρμογή [Streamlit](https://streamlit.io) που βοηθά εκπαιδευτικούς να κατανέμουν τους μαθητές/ριες της Α' Δημοτικού σε τμήματα, βασισμένη σε κοινωνικά και παιδαγωγικά κριτήρια.

## 🔍 Τι κάνει η εφαρμογή
- Υποστηρίζει εισαγωγή δεδομένων από αρχείο Excel
- Λαμβάνει υπόψη:
  - Φύλο
  - Ιδιαιτερότητες (μαθησιακές ή κοινωνικές)
  - Ζωηρότητα
  - Γλωσσική επάρκεια
  - Φιλικές σχέσεις (ζεύγη & τριάδες)
  - Συγκρούσεις & ισορροπία
- Παράγει κατανομή μαθητών σε 2+ τμήματα με ισορροπία
- Εξάγει την τελική κατανομή σε Excel
- Παράγει συνοδευτικό έντυπο υπεύθυνης δήλωσης σε Word/PDF

## 🚀 Δες την εφαρμογή Live
👉 [https://katanomi.streamlit.app](https://katanomi.streamlit.app)

## 🛠️ Εκτέλεση τοπικά
```bash
pip install -r requirements.txt
streamlit run katanomi_app.py
```

## 📁 Περιεχόμενα
- `katanomi_app.py` – Κύριο αρχείο εφαρμογής
- `requirements.txt` – Εξαρτήσεις
- `README.md` – Αυτό το αρχείο
